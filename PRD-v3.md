- **日期**：2026-06-03

## 2. 产品定义

### 2.1 一句话定位
AI Editor RSP = 印度 Viral Edit 风格的 AI 视觉模板站，提供 **AI Prompt + CapCut 模板 + Lightroom 预设**，让用户一键复制网红效果。

### 2.2 "RSP" 说明
- ✅ RSP 可安全使用（非品牌/技术名词/AI模型）
- 在社交媒体语境中，RSP = "Reels/Snaps/Posts" 的缩写，泛指短视频/图片编辑风格
- 也可理解为 "Ready Style Presets"（就绪风格预设）

### 2.3 ICP（理想客户画像）

| 用户类型 | 占比 | 痛点 | 使用场景 |
|---------|:---:|:---|:---|
| Instagram Reels 创作者 | 40% | 想做出 viral 效果但不懂复杂编辑 | 复制热门 Reels 风格 |
| 婚礼/活动摄影师 | 25% | 需要批量调色，追求印度传统美学 | 婚礼预设、节日滤镜 |
| TikTok/Shorts 创作者 | 20% | 需要快速产出高质量短视频 | CapCut 模板一键套用 |
| 照片编辑爱好者 | 15% | 想要专业效果但不想学 Lightroom | 下载预设直接应用 |

## 3. 核心内容矩阵

### 3.1 内容类型

每种模板包含三种交付物：
1. **AI Prompt** — 用于 Midjourney/Leonardo/ChatGPT 生成风格化图片
2. **CapCut Template** — 视频编辑参数/模板链接/关键帧设置
3. **Lightroom Preset** — DNG/XMP 预设文件下载

## 5. 技术架构

### 5.1 前端
- React + TypeScript + Vite
- Tailwind CSS（深色主题）
- 响应式设计（移动端优先）
- 多语言：English / Hindi / Tamil

### 5.2 后端/部署
- Cloudflare Pages（静态托管）
- Cloudflare Workers（API/表单处理）
- KV Storage（模板数据、使用统计）

### 5.3 内容管理
- 模板数据存储在 JSON/TS 文件中（首版）
